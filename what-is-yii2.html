What is Yii2?

Yii 2.0 is a high-performance, component-based PHP framework. As we know a PHP framework is a platform to create PHP web applications that provide libraries of code or functions for common tasks for a faster, secure, efficient, and easy development process of the robust and modern web application.

The framework’s name, Yii, means “simple and evolutionary” in Chinese. It also stands for “Yes, It Is!” and pronounced Yee or [ji:]

Version history -
Yii has been launched in December 2008. There are two main version of the Yii - 1.1 and 2.0. The new version 2.0 is a complete rewrite of Yii launched in October 2014 which uses the lates technologies and protocols of PHP including Composer, PSR, namespace, traits, and many more. The current stable release of Yii is 2.0.45

Requirements and prerequisites -
Yii 2.0 requires PHP 5.4.0 or above and works(recommended) best with latest version of PHP 7.

Database support -
It has database support for SQLite, MySQL, PostgreSQL, MSSQL, or Oracle databases

Licensing -
Yii is released under the fix terms of BSD License. It is free software so you can use it freely to develp a modern web applicaitons. The Yii documentation is licensed to the public under the GNU Free Documentation License (GFDL) so you can copy, distribute and/or modify it.

Application Architecture -
Yii applications architecure is based on the Model View Controller(MVC) design pattern. The Model part is used for storing the business logic and rules, and application data.The view is act as presendtiaon layer which shows the data passed by model. The controllers take user input from view and convert it to in command for passing it to Model.

You can install Yii using the "Yii2 installation using composure method" help article.

Benefits and features -
The following are Yii framwork benefits and features :

Rapid development -
In Yii you can start, run and launch the applicaiton in withing miniutes.

Architecture -
It implements MVC(Model View Controller) design pattern for better code organization.

Documentation -
The Yii guide and documentation is well written and provides easy learning process for different aspects of the Yii framework.

Ready-to-use features -
It provides many inbuilt ready made codes or features e.g. - Gii code generator, query builders and ActiveRecord, RESTful API development support, multi-tier caching support etc.

Third party supports -
The Yii is very extensible for customisation and code management and support to run third-party code within Yii applicaiton.

Powerful security practices -
The Yii maintains and practices powerful security measures for example - bcrypt password hashing, encryption, authentication, prevent SQL injection, XSS, and CSRF attacks.

Efficient caching system to speed up web app -
It uses different types of powerful and efficient caching pattern to speed up web applicaiton performance e. g. - data caching, fragment caching, page caching, and HTTP caching.

Yii framework community -
There are multiple medium are availble for Yii framework community e.g. - forum, facebook group, hall of fame, badges etc. For live chat - the Yii community offers instant help or support either via Slack or IRC channel.

Best for -
It can be used for developing different types of web applicaiton that uses PHP as scripting language. It is very suitable for developing the large-scale applications - e. g. web portals, content management systems(CMS), blog, forum, e-commerce web project, RESTful web services etc.
